## 진짜 사용 알고리즘
> 플루이드 와샬
>> 최저 거리만 구하는게 아닌 연결 범위까지 구해줄 수 있다.
<br/>
원소의 앞에 연결되어있는지, 반드시 뒤에와야 하는지를 구해준다.
<br/>
<br/>
만약에 i->j로 연결되어 있다고 한다면,  i->j / j->k 로 연결된 경우도 i->k로 연결된 것이다.
따라서 3중 for문을 돌면서 모든 노드의 우선순위를 만들어 준다. 원래 이쯤에서 max나 min을 돌리면서 노드사이 거리를 구하는 문제지만 여기서는 우선순위만 구하면 되므로 bool타입으로 해주고 구한다.
<br/>
<br/>

```c++
//플루이드-와샬 알고리즘

//중간이 가장 먼저. 해당 노드를 중심으로 앞뒤를  확인해 줘야 하기 떄문
for(int mid = 1; mid <= N; mid++){
    for(int i=1;i<=N;i++){
        for(int j=1;j<=N;j++){
            //둘다 연결되어있나 확인
            if(arr[i][m] && arr[m][j]) arr[i][j] = true;
            //만약 i->m 이고 , m->j라면 i는 j에 우선한다.
        }
    }
}
```
